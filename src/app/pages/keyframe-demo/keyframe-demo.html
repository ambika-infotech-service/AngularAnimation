How keyframes() works 

.kf-anim-bounce {
  animation: kfBounce 700ms ease-in-out;
}

@keyframes kfBounce {
  0%   { transform: translateY(0); }
  30%  { transform: translateY(-30px); }
  60%  { transform: translateY(0); }
  75%  { transform: translateY(-12px); }
  100% { transform: translateY(0); }
}
